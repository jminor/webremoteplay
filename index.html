Web Remote Play Demo

Web Remote Play allows you to play local multiplayer games remotely. That is, a game that expects multiple players using a single computer with multiple gamepads, or keyboard controls, can be played remotely over the Internet instead.

Demos

  1. Open one of the demos below.
  2. An auto-generated ID will appear in the URL.
  3. Copy paste the URL into a second browser window, or send it to your friends.

Note: The game itself runs only in the first player's browser. All other players will see a live-stream video of the game, and their inputs will be sent back to the first player's browser. Depending on the game, you may need to remap the inputs or find which keys correspond to which player.
